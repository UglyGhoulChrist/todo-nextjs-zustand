# ToDo on NextJS + Zustand

## [Проект на платформе Vercel](https://...)

**Список задач** — позволяет: 

- добавлять задачи, 
- отмечать задачу выполненной,
- возобновлять задачу,
- удалять задачу,
- удалять все задачи,
- удалять выполненные.

Хранит задачи в LocalStorage браузера.

### Технологии, применённые в проекте:

- NextJS
- Zustand
